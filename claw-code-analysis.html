Kiến trúc tổng quan

Claw Code là một agent harness framework, không phải AI

Định nghĩa quan trọng: Claw Code KHÔNG phải là một AI. Đây là harness framework — bộ khung điều phối mà Claude Code dùng để nhận lệnh, chọn tools, quản lý session, và chạy agent loop. Phần gọi API thực sự là bạn tự implement.

🎯 Mục tiêu thiết kế
  • Tool wiring — kết nối tools vào agent loop
  • Command routing — định tuyến prompt đến đúng handler
  • Session management — lưu/load trạng thái hội thoại
  • Permission system — kiểm soát quyền truy cập tools
  • Multi-turn loop — vòng lặp nhiều lượt tương tác
🧱 Nguyên tắc thiết kế
  • frozen=True dataclasses — immutable, thread-safe
  • @lru_cache — load data 1 lần duy nhất
  • Snapshot-based — tools/commands lưu trong JSON tĩnh
  • PortingModule — đơn vị thống nhất cho mọi module
  • Clean-room rewrite — không copy code gốc

6 Core Modules quan trọng nhất

Thứ tự nên đọc để hiểu codebase

# File Class / Func chính Vai trò Đọc trước
1 models.py PortingModule, UsageSummary Data classes cốt lõi — nền tảng của mọi thứ Bắt đầu đây
2 tools.py load_tool_snapshot(), PORTED_TOOLS Load 184 tools từ JSON, filter theo permissions
3 commands.py load_command_snapshot(), PORTED_COMMANDS Load 207 commands từ JSON, filter và lookup
4 query_engine.py QueryEnginePort, submit_message() Trái tim — vòng lặp turn-by-turn, state, token tracking Quan trọng nhất
5 runtime.py RuntimeSession, PortRuntime Snapshot đầy đủ 1 phiên làm việc, bootstrap session
6 main.py build_parser(), main() Entry point CLI — glue kết nối tất cả Đọc cuối 
# Ví dụ: cách QueryEnginePort hoạt động
from src.query_engine import QueryEnginePort

engine = QueryEnginePort.from_workspace()
result = engine.submit_message(
    prompt="Tìm tất cả file Python có bug",
    matched_commands=("grep",),
    matched_tools=("GrepTool", "FileReadTool"),
)
print(result.output)         # formatted output
print(result.stop_reason)    # 'completed' | 'max_turns_reached' | 'max_budget_reached'
print(result.usage)          # UsageSummary(input_tokens=..., output_tokens=...)

# Lưu session để tiếp tục sau
path = engine.persist_session()  # → ~/.claw-code/sessions/<uuid>.json
